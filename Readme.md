# Test project for Screenz

### Prerequisites
Mongodb server
Redis server

### To run:
* Clone the repository into `screenz` subdir
* `cd screenz`
* run `npm install`
* Optionally edit .env.example to provide various parameters
* run `npm run worker &`
* run `npm start`


[API Documentation and testbed (Swagger)](http://screenz.shprota.com/api-docs)
